# Inspiration and reference

References for the on-demand vote feature (yes/no UI, keybinds).

## External (links)

### CyclePresetPlugin (nvrlift)

- **Source:** https://github.com/nvrlift/CyclePresetPlugin  
- Track voting for AssettoServer; reference for preset/vote structure.

### comfy weather vote (Tuttertep)

- **Source:** https://overtake.gg/downloads/comfy-weather-vote.61303  
- Lua app: AssettoServer weather voting as **buttons** instead of chat.  
- **Use:** UI inspiration – button-based voting; mimic for “Change preset? Yes / No” and keybinds.

### comfy admin (Tuttertep)

- **Source:** https://www.overtake.gg/downloads/comfy-admin.57641/  
- Admin UI including weather voting buttons. Same idea as above.

### VotingPresetPlugin (upstream)

- **Source:** AssettoServer (we forked it into this repo).  
- Timer-based preset voting; we keep that and add on-demand votes with in-game yes/no UI.
